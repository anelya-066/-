# -Панель специальных возможностей (Accessibility Toolbar):

Регулировка размера шрифта: Мгновенное масштабирование контента (4 уровня размера) без разрушения структуры интерфейса.

Режим высокой контрастности: Специальная высококонтрастная черно-желтая тема для пользователей с нарушениями зрения (соответствует базовым принципам доступности Web Content Accessibility Guidelines - WCAG).Inclusive Learning — Образовательная платформа для детей с особыми потребностями

«Inclusive Learning» — это современное, адаптивное и доступное веб-пространство, разработанное для поддержки, обучения и социализации детей с особыми образовательными потребностями (включая РАС, СДВГ, ЗПР, логопедические нарушения и нарушения зрения).

Платформа создана с учетом жестких стандартов доступности и обеспечивает равные возможности для получения знаний, вовлечения в игровой учебный процесс и взаимодействия между родителями, специалистами и детьми.

🌟 Ключевые возможности проекта



Интерактивный каталог программ (с фильтрацией):

Динамическая фильтрация адаптированных курсов по возрастным категориям (3-6 лет, 7-11 лет, 12-16 лет).

Каждая карточка содержит метки особенностей развития (РАС, СДВГ, ЗПР, ДЦП и т.д.).

Интерактивные модальные окна:

Полностью бесшовные всплывающие окна для записи на курсы, регистрации личного кабинета и бронирования консультаций со специалистами.

Интерактивный просмотр подробных статей и методических рекомендаций прямо на странице («ридер-режим»).

Имитация серверного ответа (Кастомные Toast-уведомления):

Никаких раздражающих системных окон alert(). Формы используют плавные кастомные всплывающие сообщения об успешной отправке.

Адаптивный дизайн:

Платформа полностью оптимизирована для мобильных устройств, планшетов и экранов с высоким разрешением. Все элементы управления и touch-зоны увеличены для удобства детей с особенностями моторики.
